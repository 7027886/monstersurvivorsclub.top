About The Game

Monster Survivors is an exciting survival action game where players face endless waves of monsters in a dynamically evolving world.

With each run, players can unlock new abilities, discover powerful combinations, and challenge themselves to survive longer against increasingly difficult enemies.

Key Features:

  • Roguelike survival gameplay
  • Multiple character abilities
  • Progressive difficulty system
  • Unique power-up combinations
